# Varejão Gen

Sistema simples de console para gerenciar uma lista de frutas, desenvolvido como exercício de lógica de programação em Python.

## Funcionalidades
* Cadastro de novos itens via input do usuário.
* Listagem numerada para facilitar a visualização.
* Exclusão de itens específicos utilizando o índice da lista.

## Conceitos Aplicados
* Manipulação de listas: `append()` e `pop()`.
* Estruturas de repetição: `while` e `for`.
* Funções integradas: `enumerate()` para exibição de índices.
